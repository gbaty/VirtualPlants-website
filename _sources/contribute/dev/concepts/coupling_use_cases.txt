Package Views, Helper
---------------------

Several widgets provide a view to package manager. For instance:

    - :obj:`~openalea.oalab.package.widgets.PackageViewWidget`
    - :obj:`~openalea.oalab.package.widgets.PackageCategorieViewWidget`
    - :obj:`~openalea.oalab.package.widgets.PackageSearchWidget`

Each package, node or data provide a documentation. We want to display this
documentation on a special place, the HelpApplet widget, if available.
We do not want to force use of this applet. Package View can run without it.
